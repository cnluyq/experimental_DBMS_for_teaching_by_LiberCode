# 实验指南

本目录包含一系列实验教程，帮助你逐步构建和深入理解ProjoDB各组件。每个实验都有明确的目标、步骤和预期结果。

## 📚 实验列表

### 初级实验

1. **[实验1：缓冲区池性能分析](exp1_buffer_analysis.md)**
   - 理解LRU算法的工作原理
   - 测量不同工作负载下的缓存命中率
   - 分析帧数对性能的影响

2. **[实验2：实现页位图分配器](exp2_page_allocator.md)**
   - 从零实现页分配器
   - 学习位图数据结构
   - 测量分配和释放性能

### 中级实验

3. **[实验3：B+树索引实现](exp3_bplus_tree.md)**
   - 实现B+树的插入、搜索、删除
   - 学习平衡树结构
   - 测试范围查询

4. **[实验4：WAL预写日志](exp4_wal.md)**
   - 实现Write-Ahead Logging
   - 学习崩溃恢复算法
   - 测试原子性保证

### 高级实验

5. **[实验5：事务管理](exp5_transactions.md)**
   - 实现ACID特性
   - 构建锁管理器
   - 处理死锁

6. **[实验6：查询执行引擎](exp6_query_engine.md)**
   - 设计物理算子（表扫描、选择、投影）
   - 实现火山模型迭代器
   - 执行简单SQL查询

## 🎯 实验目标

每个实验完成后，你将：

1. **理解核心概念**：清楚该组件的设计原理和算法
2. **完成代码实现**：至少达到基本功能的可运行版本
3. **通过测试用例**：验证实现的正确性
4. **进行性能分析**：理解不同配置的影响
5. **撰写实验报告**：总结学习心得和改进建议

## 🔧 实验环境准备

### 1. 克隆项目
```bash
git clone <repo-url>
cd projo
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 熟悉代码结构
```bash
# 查看目录树
tree src/ -L 2

# 查看现有测试
ls tests/
```

### 4. 运行基准测试（可选）
```bash
# 确保环境正常
python -m pytest tests/test_buffer.py -v
```

## 📝 实验提交

每个实验完成后，请：
1. 更新代码到对应目录
2. 编写实验报告（Markdown格式）
3. 提交到实验目录：`docs/tutorials/expN_你的名字.md`

## 💡 学习建议

- **循序渐进**：按顺序完成实验，前面的是后面的基础
- **动手实践**：不要只看不写，每个实验都要动手编码
- **调试技巧**：善用打印日志、断点调试
- **查阅资料**：多参考教材、开源实现、技术博客
- **交流讨论**：与同学讨论，参加学习小组

## 🆘 遇到困难？

1. **阅读现有代码**：`src/core/buffer.py`有完整的LRU实现可供参考
2. **查看测试用例**：`tests/`目录有各模块的测试
3. **搜索问题**：用Google/Stack Overflow搜索错误信息
4. **拆分问题**：大问题分解为小步骤，逐步解决
5. **求助社区**：在GitHub Issues中提问
